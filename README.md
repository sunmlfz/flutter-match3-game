## 功能特性

### 核心玩法
- ✅ 标准 Match-3：点击选中 → 点击相邻交换 → ≥3连消除
- ✅ 自动下落 & 连锁消除（Cascade）
- ✅ 只允许产生消除的交换

### 特殊元素
| 生成方式 | 效果 |
|---------|------|
| 4连（横） | 🌟 横向直线消除全行 |
| 4连（纵） | 🌟 纵向直线消除全列 |
| T/L形 | 💥 3×3范围爆炸 |
| 5连 | 🌈 全屏同色消除 |

### 障碍物系统
- 🧊 **冰块**：相邻消除可破坏（1-2层）
- 🪨 **石块**：不可移动，需特殊元素破坏
- ⛓️ **锁链**：解锁后才能移动元素
- 🟩 **粘液**：每回合扩散

### 道具系统
- 🔨 **锤子**：点击消除单个元素
- 💣 **炸弹**：3×3范围消除
- 🔄 **重排**：重新打乱棋盘

### 关卡系统
| 关卡 | 名称 | 目标 | 限制 |
|-----|------|------|------|
| 1 | 初识消消乐 | 消除30个红色 | 20步 |
| 2 | 寒冰之地 | 清除10个冰块 | 25步 |
| 3 | 石之迷宫 | 5000分 + 清石块 | 30步 |
| 4 | 极速消除 | 8000分 | 60秒 |
| 5 | 终极消消乐 | 冰块+石块+分数 | 40步 |

### 评分系统
- 基础：每消除1个 = 50分
- 连击加成：连击数 × 0.5
- 连锁加成：连锁级数 × 0.3
- 特殊元素触发：+200分
- 剩余步数奖励：每步 × 100分
- ⭐️ 1-3星评级
